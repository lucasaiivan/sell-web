# Tech Stack & Versiones

## Core
*   **Flutter:** Stable Channel (Latest).
*   **Dart:** 3.x (Aprovechando Patterns, Records, Sealed classes).

## Backend (Firebase)
*   **Hosting:** Para despliegue web.
*   **Firestore:** Base de datos NoSQL principal.
*   **Authentication:** Gestión de usuarios.
*   **Cloud Functions:** Lógica de negocio crítica en servidor (TypeScript/JS).
*   **Analytics:** Seguimiento de eventos.

## Librerías Clave
*   **State Management:** `provider`
*   **Routing:** `go_router` (si aplica) o Navigator 2.0.
*   **UI:** `flutter_animate` (para efectos), `google_fonts`.
*   **Utils:** `intl` (para fechas y monedas), `json_annotation`.
