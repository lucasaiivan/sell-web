# Components

Este directorio contiene componentes visuales genéricos y reutilizables.

## Contenido
- `avatar_product.dart`: Avatar para productos.
- `avatar_user.dart`: Avatar para usuarios.
- `dividers.dart`: Divisores personalizados.
- `image.dart`: Componentes de imagen.
- `progress_indicators.dart`: Indicadores de carga.
- `responsive_helper.dart`: Utilidades para diseño responsivo.

## Uso
Importar desde `core_widgets.dart` o directamente desde `components.dart`.
